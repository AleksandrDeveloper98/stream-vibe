# Stream Vibe

Кинопоиск проект — платформа для поиска и просмотра фильмов и сериалов.

## 📋 О проекте

Stream Vibe — это веб-приложение, созданное для удобного поиска и просмотра контента. Проект разработан с акцентом на производительность, адаптивность и чистоту кода.

### ✨ Функциональность
- 🎬 **6 уникальных страниц** с разнообразным контентом
- 🔄 **Плавная навигация** между страницами 
- 🎥 **Встроенное воспроизведение** видео
- 📱 **Полная адаптивность** под все типы устройств (desktop, tablet, mobile)
- 📝 **Маска ввода** для форм с использованием IMask
- 🧩 **Чистая архитектура** по методологии БЭМ

## 🛠 Технологии

- **Minista** — сборщик проекта на основе Vite
- **HTML5** — семантическая верстка
- **SCSS** — препроцессор для стилей
- **БЭМ** — методология именования классов
- **JavaScript (ES6+)** — логика приложения
- **Swiper** — touch-слайдеры
- **IMask** — маски для полей ввода
- **classnames** — условное объединение CSS-классов
